## Naming and hygiene

- Use deterministic names and avoid timestamp-only filenames.
- Use public demo content in all screenshots.
- Keep image files relatively small and single-purpose.
- If a screenshot is sensitive and can not be sanitized, keep only the TODO entry and do not add the image.
