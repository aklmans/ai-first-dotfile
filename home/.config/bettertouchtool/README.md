## BetterTouchTool Gesture Presets

This directory keeps only a small, reproducible preset script:

- `aerospace-gestures.sh` recreates AeroSpace workspace swipe/workspace-related gestures through BetterTouchTool AppleScript.

Install flow:

1. Run `bootstrap/install/bettertouchtool.sh` to install/apply the preset.
2. Confirm trackpad permissions are enabled in macOS System Settings > Privacy & Security > Accessibility.
3. If you want custom gestures, edit this preset script and re-run the installer.

For safety, full BetterTouchTool application database files are not tracked in this repository.
